Handtris

핸드 모션을 활용한 실시간 1:1 대결 테트리스 게임

💡담당한 기능

  • WebSocket `STOMP` 프로토콜을 사용하여 실시간성 보장: 테트리스 블록의 좌표값, 컬러 등의 정보를 Text형식으로 상대방 유저에게 전달
  • Redis 캐시를 이용하여 API 호출 시간 단축: 게임방과 유저 정보를 매핑해 저장, 유저가 나가면 Redis에서 정보 삭제
  • ChannelInterceptor를 이용해 요청 전처리: 메시지를 가로채 요청 확인 및 수정, Interceptor에서 상대방 정보를 Header에 담아 해당 API에 전달
  • Junit과 Mockito를 이용한 유닛 단위 테스트 작성
